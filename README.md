## RPG Framework

Low Poly為主視覺的開放世界角色扮演遊戲
以4人以下的合作模式進行遊戲
並沒有主要的遊戲玩法
會以程序性生成的無限地圖讓玩家探索
締造獨一無二的冒險故事

已承貸80萬元新台幣進行開發
礙於專案進度落後
無法如期上市獲利
這個專案將保留到業餘時間研究
